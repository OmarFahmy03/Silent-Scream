## 🧠 About The Project

Silent Scream is a horror game project developed using Unity and C#.
The game focuses on atmosphere, tension, and immersive gameplay mechanics rather than relying solely on jump scares.

This project demonstrates:

Game logic architecture in C#

Custom gameplay systems

Scene management

Horror atmosphere design

Audio and lighting integration

## 🎮 Gameplay Overview

Explore a dark and mysterious environment

Solve environmental interactions

Experience psychological tension

Survive through observation and awareness
